Holistic ML Analysis and Mozilla Evaluation

This section is about our machine learning analysis and assessment of a dataset called the "Mozilla not found" dataset, which contains information on specific IoT devices and their privacy policies. The analysis involved extracting various features from the policies, such as coherence score, entropy, reading level, and others, to gain insights into the privacy policies of these devices. The analysis also involved a keyword analysis of these policies to obtain values related to Opt-Out, Data, Data Security, First-party collection, and others, which were used to create the final dataset. To achieve high accuracy in the classification model, the analysis opted for logistic regression, which proved to be highly effective. The resulting dataset was created based on the "like'' or "dislike" ratings of the devices' privacy policies evaluated against the Mozilla not found project's standards. The analysis provides valuable insights into the privacy policies of these IoT devices, which could be useful in improving the compliance of these devices with privacy standards.
